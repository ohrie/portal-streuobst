# Mitmachen am Streuobstwiesen Portal

Vielen Dank für dein Interesse, zu diesem Projekt beizutragen!

## Wie du mitmachen kannst

### Kartendaten verbessern

Die wichtigste Möglichkeit zur Mitarbeit ist das Einpflegen und Verbessern von Streuobstwiesen in **OpenStreetMap** – dem offenen Kartenprojekt, auf dem unsere Daten basieren. Eine Anleitung findest du auf der [Mitmachen-Seite](https://portal-streuobst.de/data/) des Portals.

### Code-Beiträge

Code-Beiträge sind willkommen, sofern sie den Scope der Website entsprechen. Wenn möglich bespreche deine Idee zuerst in einem Issue bevor du einen Pull-Request erstellst.

1. **Fork** das Repository auf GitHub.
2. Erstelle einen neuen Branch: `git checkout -b feature/mein-feature`
3. Nimm deine Änderungen vor und committe sie: `git commit -m 'Beschreibung der Änderung'`
4. Push deinen Branch: `git push origin feature/mein-feature`
5. Öffne einen **Pull Request**.

### Bugs melden & Ideen einreichen

Nutze die [GitHub Issues](https://github.com/ohrie/portal-streuobst/issues) für:
- Fehlerberichte (Bug Reports)
- Feature-Wünsche

## Entwicklungsumgebung aufsetzen

```bash
npm install
npm run dev
```

Die App ist dann unter [http://localhost:3000](http://localhost:3000) erreichbar.
